# 🛒 Catálogo Interactivo de Productos

---

## 📋 1. Product backlog (Historias de usuario)

* **HU-01:** Como usuario, quiero ver una lista de productos con su imagen, precio y descripción para conocer la oferta disponible.
* **HU-02:** Como usuario, quiero buscar productos por nombre o categoría para encontrar rápidamente lo que necesito.
* **HU-03:** Como usuario, quiero agregar productos a un carrito de compras para gestionar mi selección antes de adquirirla.
* **HU-04:** Como usuario, quiero ordenar los productos por precio (menor a mayor y viceversa) para optimizar mi presupuesto.
